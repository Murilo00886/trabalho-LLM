## Como Usar a Aplicação

1. Inicie o Servidor: No terminal da pasta do projeto, rode o comando `npm run dev`. Você verá a mensagem informando que o servidor está online.
2. Abra no Navegador: Acesse o endereço `http://localhost:3000` no seu navegador de preferência.
3. Faça sua Pergunta: Na caixas de texto que vai aparecer na tela, digite qualquer dúvida de programação, lógica ou peça ajuda com algum erro que esteja travando o seu código.
4. Envie e Pratique: Clique no botão "Enviar Dúvida". O "Professor" vai te responder explicando conceitos e dando exemplos.
